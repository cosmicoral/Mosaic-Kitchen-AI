# Mosaic-Kitchen-AI
AI-powered multicultural meal planning and grocery assistant for diverse households in the UK.
Vision

Mosaic Kitchen helps multicultural households in the UK make healthier, cheaper, and more culturally relevant food decisions.

Unlike traditional meal planning apps, Mosaic Kitchen considers:

Cultural food preferences
Asian grocery ecosystems
Food waste reduction
Expiration date tracking
Grocery optimisation
AI-powered meal recommendations
